Angular Skeleton

Ce squelette est équipé des meilleures pratiques et outils pour démarrer rapidement le développement d'une application Angular.
Caractéristiques Principales
  • Intercepteur de Token pour la gestion des authentifications
  • Snackbar Service pour les notifications utilisateur
  • Auth Facade pour simplifier les flux d'authentification
  • Lazy Loading pour une meilleure performance
  • Store Elf pour la gestion d'état
Guide de Démarrage Rapide
  1. Clonez le dépôt : git clone url_du_dépôt
  2. Installez les dépendances : npm install
  3. Lancez le serveur de développement : ng serve
  4. Visitez http://localhost:4200/ pour voir l'application en action.
Personnalisation

Pour adapter ce squelette à votre projet personnel, suivez ces étapes :

  1. Modifiez les variables d'environnement dans les fichiers environment.ts et environment.prod.ts
  2. Adaptez les modules, composants, services et façades selon les besoins de votre application.
  3. Utilisez le module Angular CLI pour ajouter ou modifier les éléments de votre application.
