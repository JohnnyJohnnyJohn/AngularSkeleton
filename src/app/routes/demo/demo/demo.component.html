Démonstration des Composants

App Button

Les boutons sont des éléments interactifs qui permettent aux utilisateurs d'effectuer des actions.

Il existe plusieurs types de boutons :

  • Basic : bouton de base
  • Raised : bouton en relief
  • Stroked : bouton avec bordure
  • Flat : bouton plat

Il est possible de désactiver un bouton en ajoutant l'attribut disabled.

App Icon Button

Les boutons icones sont des éléments interactifs qui permettent aux utilisateurs d'effectuer des actions.

Il existe plusieurs types de boutons icones :

  • Icon : bouton icone
  • Fab : bouton icone flottant
  • Mini Fab : bouton icone flottant mini
  • Extended Fab : bouton icone flottant étendu

Il est possible de changer la couleur d'un bouton icone en ajoutant l'attribut color.

App Input Text

Les champs de texte permettent aux utilisateurs d'entrer des données.

Il existe plusieurs types de champs de texte :

  • Outline : champ de texte avec bordure
  • Filled : champ de texte rempli

Il est possible de rendre un champ de texte obligatoire en ajoutant l'attribut isRequired.

Il est possible de changer la couleur d'un champ de texte en ajoutant l'attribut colorValue.

Il est possible de changer l'apparence d'un champ de texte en ajoutant l'attribut appearanceValue.

Il est possible de changer le placeholder d'un champ de texte en ajoutant l'attribut placeholderValue.

Il est possible de changer l'autocomplétion d'un champ de texte en ajoutant l'attribut autocompleteValue.

Il est possible de changer le message d'erreur d'un champ de texte en ajoutant l'attribut errorMessage.
